FULL SETUP

1. Python (3.9+)
   Grab it → https://www.python.org/downloads/
   Got it already? Chill.

2. Clone & Dive In
   git clone https://github.com/arandomusernameyesaveryrandomusername/RSACodec-Roblox-Sine-Audio.git
   cd RSACodec-Roblox-Sine-Audio

3. Virtual Env (do it like a pro)

   Windows PowerShell
   python -m venv venv
   .\venv\Scripts\Activate.ps1

   Windows CMD
   python -m venv venv
   .\venv\Scripts\activate.bat

   Linux / macOS
   python3 -m venv venv
   source venv/bin/activate

   (see (venv) prefix? Good proceed to next part! If not, make sure you created the venv properly by looking if theres a venv folder adn that you ran the right activate command look at your teminal carefully...)

4. Install the goods
   pip install --upgrade pip
   pip install numpy scipy

   Turbo mode (optional):
   pip install numba

5. Encode the audio 🔥
   python rsc_encoder.py

   → slap your message in
   → spits out a sneaky .wav
   → upload to Roblox (6–20 seconds hits perfect)

To Decode:
   python rsc_decoder.py
